🎨 HTML, CSS & Tailwind CSS

HTML — El Esqueleto

HTML (HyperText Markup Language) es el lenguaje que usamos para estructurar una página web. Piensa en él como el esqueleto de un cuerpo humano: define dónde va la cabeza (head), el cuerpo (body), los brazos (header, nav), el torso (main) y las piernas (footer).

En nuestro proyecto, cada página es un archivo HTML. Por ejemplo, index.html tiene esta estructura básica:



  
    
    Mi Página
    
  
  
    
...

    
...

    
...

Las etiquetas HTML más usadas en el proyecto:

  • — Un contenedor genérico (como una caja)
  • — Una sección temática
  • — Navegación (menús)
  • — Enlaces (hipervínculos)
  • — Imágenes
