🏆 Página de Eliminatorias

¿Qué hace esta página?

Muestra el árbol de eliminatorias (también llamado bracket) del torneo. Es la representación visual de cómo los equipos avanzan desde Octavos de Final hasta la Gran Final.

La página tiene un diseño de 3 columnas con conectores gráficos entre las rondas, similar a lo que ves en la televisión durante el Mundial.

Cómo se Construye el Bracket

El proceso tiene 3 pasos principales:

  1. Obtener datos — Llamamos a la API para cada ronda: LAST_16, QUARTER_FINALS, SEMI_FINALS, THIRD_PLACE, FINAL
  2. Mapear partidos — Asignamos cada partido a una posición en el bracket usando data-match-id (del 0 al 12)
  3. Rellenar tarjetas — La función fillMatchCard() pone los datos en cada tarjeta HTML
const [last16Data, quarterData, semiData, thirdData, finalData] = await Promise.all([
  getMatchesByStage('LAST_16'),
  getMatchesByStage('QUARTER_FINALS'),
  // ...
]);
// Mapear a posiciones del bracket
const bracketMatches = {
  0: leftLast16[0],   // Octavos izquierda #1
  4: leftQuarters[0], // Cuartos izquierda #1
  8: finalMatch,       // Final
  // ...
};

fillMatchCard() — Rellenar Tarjetas

Esta función recibe una tarjeta HTML y un partido de la API, y actualiza todo el contenido:

  • Información — Número de partido y fecha
  • Estado — Color y texto según si está programado, en vivo o finalizado
  • Equipos — Escudos y nombres traducidos al español
  • Marcador — Goles (con manejo de penaltis)
  • Winner highlighting — El equipo perdedor se atenúa (clase opacity-60)
// Ejemplo: atenuar al perdedor
if (homeScore > awayScore) {
  awayDiv.classList.add('opacity-60'); // Visitante pierde
} else if (awayScore > homeScore) {
  homeDiv.classList.add('opacity-60'); // Local pierde
}

Promise.all — Carga en Paralelo

Para obtener todas las rondas del torneo usamos Promise.all que ejecuta 5 peticiones a la vez. Esto es mucho más rápido que hacerlas una tras otra.

const [r1, r2, r3, r4, r5] = await Promise.all([
  fetch('/api/ronda1'),
  fetch('/api/ronda2'),
  fetch('/api/ronda3'),
  fetch('/api/ronda4'),
  fetch('/api/ronda5')
]);

🏆 Knockout Page

What does this page do?

Shows the knockout bracket of the tournament. It's a visual representation of how teams advance from the Round of 16 to the Grand Final.

The page has a 3-column layout with graphic connectors between rounds, similar to what you see on TV during the World Cup.

How the Bracket is Built

3 steps: Get data (API calls for each round), Map matches (assign each match to a bracket position using data-match-id 0-12), Fill cards (fillMatchCard() puts data into each HTML card).

fillMatchCard()

Updates each card with: match info & date, status (color/text), team crests & translated names, scores (with penalty handling), and winner highlighting (loser gets opacity-60).
