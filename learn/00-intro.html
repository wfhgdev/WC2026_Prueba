🌍 Introducción al Proyecto

¿Qué es este proyecto?

Es una aplicación web sobre la Copa Mundial FIFA 2026. Muestra información real del torneo: partidos, resultados, clasificaciones de grupos, el árbol de eliminatorias y estadísticas de jugadores y equipos.

Los datos vienen de una API externa llamada football-data.org, que provee información actualizada del fútbol mundial.

El proyecto fue creado con fines educativos — para aprender cómo se construye una web moderna usando solo HTML, CSS, JavaScript y un poco de PHP.

Tecnologías Usadas

  • HTML5 — La estructura de las páginas (etiquetas, atributos, semántica)
  • Tailwind CSS — Un framework de CSS que usa clases pequeñas y reutilizables en lugar de escribir CSS personalizado
  • JavaScript (Vanilla) — JavaScript puro, sin frameworks como React o Vue. Usamos el DOM, fetch, async/await, localStorage
  • PHP — Un pequeño archivo proxy.php que sirve como intermediario para llamar a la API
  • API football-data.org — Una API REST que devuelve datos en formato JSON

¿Cómo funciona en resumen?

  1. El navegador carga un archivo HTML (ej: index.html)
  2. El HTML incluye Tailwind CSS desde un CDN y nuestros archivos JavaScript
  3. JavaScript llama a api.js que obtiene datos de la API (a través del proxy PHP para evitar el problema CORS)
  4. Los datos se guardan en localStorage (caché) para no llamar la API cada vez
  5. Las funciones de cada página toman esos datos y crean elementos HTML dinámicamente (tarjetas de partidos, tablas de grupos, etc.)
  6. Todo el texto de la interfaz está en español gracias al sistema de internacionalización (i18n)

🌍 Project Introduction

What is this project?

It's a web application about the FIFA World Cup 2026. It shows real tournament information: matches, results, group standings, the knockout bracket, and player/team statistics.

The data comes from an external API called football-data.org, which provides up-to-date football information worldwide.

This project was created for educational purposes — to learn how to build a modern website using only HTML, CSS, JavaScript, and a little PHP.

Technologies Used

  • HTML5 — The page structure (tags, attributes, semantics)
  • Tailwind CSS — A CSS framework that uses small, reusable classes instead of writing custom CSS
  • Vanilla JavaScript — Pure JavaScript, no frameworks like React or Vue. We use the DOM, fetch, async/await, localStorage
  • PHP — A small proxy.php file that acts as an intermediary to call the API
  • football-data.org API — A REST API that returns data in JSON format

How does it work (summary)?

  1. The browser loads an HTML file (e.g. index.html)
  2. The HTML includes Tailwind CSS from a CDN and our JavaScript files
  3. JavaScript calls api.js which gets data from the API (through the PHP proxy to avoid CORS issues)
  4. The data is saved in localStorage (cache) so we don't call the API every time
  5. The functions on each page take that data and create HTML elements dynamically (match cards, group tables, etc.)
  6. All interface text is in Spanish thanks to the i18n system
