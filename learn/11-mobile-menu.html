📱 Menú Móvil Compartido

¿Qué es?

Es un menú que aparece en pantallas pequeñas (móviles) cuando haces clic en el botón hamburguesa (☰). Cubre toda la pantalla y muestra enlaces a todas las secciones del sitio.

El archivo responsable es js/menu.js y se incluye en todas las páginas del proyecto.

El HTML del Menú

En cada página HTML hay un

con id="mobile-menu-overlay" que está oculto inicialmente (clase hidden). Cuando se activa, se muestra como un overlay que cubre toda la pantalla.

El JavaScript (menu.js)

El código JavaScript está envuelto en una IIFE (Immediately Invoked Function Expression). Esto significa que la función se ejecuta automáticamente en cuanto se carga, sin necesidad de llamarla.

(function() {
  'use strict';

  function initMobileMenu() {
    const openBtn = document.getElementById('open-menu');
    const closeBtn = document.getElementById('close-menu');
    const menu = document.getElementById('mobile-menu-overlay');

    // Función para mostrar/ocultar el menú
    function toggleMenu(show) {
      menu.classList.toggle('hidden', !show);
      // Bloquear scroll del body cuando el menú está abierto
      document.body.style.overflow =
        menu.classList.contains('hidden') ? '' : 'hidden';
    }

    // Abrir al hacer clic en hamburguesa
    openBtn.addEventListener('click', () => toggleMenu(true));

    // Cerrar al hacer clic en X
    closeBtn.addEventListener('click', () => toggleMenu(false));

    // Cerrar al hacer clic fuera del menú
    menu.addEventListener('click', (e) => {
      if (e.target === menu) toggleMenu(false);
    });

    // Cerrar con tecla Escape
    document.addEventListener('keydown', (e) => {
      if (e.key === 'Escape' && !menu.classList.contains('hidden')) {
        toggleMenu(false);
      }
    });
  }

  // Esperar a que el DOM esté listo
  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', initMobileMenu);
  } else {
    initMobileMenu();
  }
})();

¿Qué es una IIFE?

Una IIFE (Immediately Invoked Function Expression) es una función que se ejecuta inmediatamente después de ser definida. Se usa para:

  • Encapsular código — Las variables dentro de la IIFE no contaminan el ámbito global
  • Ejecutar código automáticamente — No necesitas llamar a la función, se ejecuta sola
// IIFE básica
(function() {
  var mensaje = 'Esto está encapsulado';
  console.log(mensaje);
})();
// mensaje NO existe fuera de la función

Body Scroll Lock

Cuando el menú está abierto, el fondo (body) no debería desplazarse. Esto se logra cambiando document.body.style.overflow a 'hidden' cuando el menú se abre, y restaurándolo a '' cuando se cierra.

document.body.style.overflow = 'hidden'; // Bloquear scroll
document.body.style.overflow = '';       // Restaurar scroll

📱 Shared Mobile Menu

What is it?

A menu that appears on small screens (mobile) when you click the hamburger button (☰). It covers the full screen and shows links to all site sections. The responsible file is js/menu.js and it's included on every page.

HTML Structure

Each HTML page has a

with id="mobile-menu-overlay" initially hidden (class hidden). When activated, it shows as a fullscreen overlay.

JavaScript (menu.js)

The code is wrapped in an IIFE (Immediately Invoked Function Expression). It handles: open (hamburger click), close (X click, outside click, Escape key), and body scroll lock.

Body Scroll Lock

When the menu is open, the background body shouldn't scroll. This is done by setting document.body.style.overflow = 'hidden' when opening, and restoring it to '' when closing.
