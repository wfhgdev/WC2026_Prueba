📊 Página de Grupos

¿Qué hace esta página?

Muestra las tablas de posiciones de la fase de grupos del Mundial. Cada grupo tiene una tabla con: posición, equipo, partidos jugados (PJ), victorias (G), empates (E), derrotas (P), goles a favor (GF), goles en contra (GC), diferencia de goles (DIF) y puntos (PTS).

IMPORTANTE: No usamos el endpoint de standings de la API. En su lugar, calculamos las tablas desde los partidos de la fase de grupos. Esto nos da más control y precisión.

calculateTeamStats() — La Función Clave

Esta función recibe un equipo y los partidos de su grupo, y calcula todas sus estadísticas:

function calculateTeamStats(teamTLA, teamCrest, matches) {
  const stats = {
    playedGames: 0,
    won: 0, draw: 0, lost: 0,
    goalsFor: 0, goalsAgainst: 0,
    points: 0
  };

  matches.forEach(match => {
    if (match.status !== 'FINISHED') return; // Solo partidos terminados

    const homeTLA = i18n.getTLA(match.homeTeam);
    const awayTLA = i18n.getTLA(match.awayTeam);
    // ¿Es este equipo local o visitante?
    const isHome = homeTLA === teamTLA;
    const isAway = awayTLA === teamTLA;
    if (!isHome && !isAway) return;

    stats.playedGames++;
    if (isHome) {
      stats.goalsFor += homeScore;
      stats.goalsAgainst += awayScore;
      if (homeScore > awayScore) stats.won++;
      else if (homeScore === awayScore) stats.draw++;
      else stats.lost++;
    }
    // ...
  });

  stats.points = (stats.won * 3) + stats.draw; // 3 pts por victoria, 1 por empate
  return stats;
}

Ordenación de la Tabla

Los equipos se ordenan por:

  1. Puntos (más alto primero)
  2. Diferencia de goles (GD = GF - GC)
  3. Goles a favor (GF)

📊 Groups Page

What does this page do?

Shows group standings for the World Cup group stage. Each group has a table with: position, team, matches played (MP), wins (W), draws (D), losses (L), goals for (GF), goals against (GA), goal difference (GD), and points (PTS).

IMPORTANT: We don't use the API's standings endpoint. Instead, we calculate the tables from matches in the group stage. This gives us more control and accuracy.

calculateTeamStats() — The Key Function

This function receives a team and its group matches, then calculates all statistics: played games, wins, draws, losses, goals for, goals against, and points (3 for win, 1 for draw).

Table Sorting

Teams are sorted by: Points (highest first), Goal difference (GD = GF - GA), Goals for (GF).
