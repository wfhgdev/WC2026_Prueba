🏠 Página de Inicio

Estructura General

La página de inicio (index.html) tiene 3 partes principales:

  • Header — Barra superior con logo, menú de navegación (escritorio) y botón hamburguesa (móvil)
  • Main — Una sección Hero (imagen grande con texto) y un Bento Grid (cuadrícula de tarjetas)
  • Footer — Información legal y enlaces secundarios

Hero Section

El Hero es la primera impresión del sitio. Tiene:

  • Una imagen de fondo grande (de la Copa Mundial)
  • Un gradiente oscuro encima para que el texto sea legible
  • El título "COPA MUNDIAL 2026: UNIDOS COMO UNO"
  • Un botón "Consigue tus Entradas"

🏠 Home Page

General Structure

The home page (index.html) has 3 main parts:

  • Header — Top bar with logo, desktop nav menu, hamburger button (mobile)
  • Main — Hero section (large image with text) and Bento Grid (card grid)
  • Footer — Legal info and secondary links

Hero Section

The Hero is the site's first impression. It has:

  • A large background image (from the World Cup)
  • A dark gradient overlay for text readability
  • The title "COPA MUNDIAL 2026: UNIDOS COMO UNO"
  • A "Get Tickets" button
