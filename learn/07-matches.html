⚽ Página de Partidos

¿Qué hace esta página?

Muestra todos los partidos del Mundial 2026 con filtros para verlos por estado: Todos, Programados, En Vivo, Finalizados.

Cada partido se muestra como una tarjeta con los escudos de los equipos, el marcador (o la hora si no ha empezado), el grupo/fase y el estadio.

Filtros

Los filtros son botones que llaman a renderMatches(estado) con el estado seleccionado. El botón activo se resalta con la clase bg-primary-fixed.

Skeleton Loaders

Mientras los datos se cargan, mostramos esqueletos (skeletons) que simulan la forma de las tarjetas con una animación de brillo (shimmer). Esto mejora la experiencia del usuario porque no ve una pantalla en blanco.

Tarjetas de Partido

Cada tarjeta se construye dinámicamente con createElement y innerHTML. Dependiendo del estado del partido, muestra:

  • En vivo — Marcador actual, minuto, barra superior de color, pulso rojo en el estado
  • Finalizado — Marcador final, posible resultado de penaltis, equipo perdedor atenuado
  • Programado — Hora del partido y fecha relativa (Hoy, Mañana, fecha)

⚽ Matches Page

What does this page do?

Shows all World Cup 2026 matches with filters by status: All, Scheduled, Live, Finished.

Each match is shown as a card with team crests, score (or time if not started), group/stage, and venue.

Filters

Filters are buttons that call renderMatches(status) with the selected status. The active button is highlighted.

Skeleton Loaders

While data loads, we show skeletons that simulate card shapes with a shimmer animation. This improves UX by avoiding blank screens.

Match Cards

Each card is built dynamically. Depending on match status it shows: Live (score + minute + pulsing red), Finished (final score + penalties), Scheduled (time + relative date).
